This is the root folder for the initiative, "Engineering Reboot 2026"



Project Objective:

&#x20; - Get back to software engineering, leveraging 22+ years of industry experience, 8 years actually 

&#x20;   spent on software development, while the rest doing project management, and software engineering

&#x20;   management.  This is a 6-month long self-directed activity which covers essential software engineering

&#x20;   principles as well as using AI as a companion for software engineering.

&#x20; - The study sessions cover various topics which should enable one to bet back to software engineering and become

&#x20;   up-to-date to the latest development practices after a long period of non-practice

&#x20; - The 6-month period is divided into 6 months of 4 weekly sprints per month, with each week taking 5 hours. 

&#x20;   Daily tasks are prepared one week in advance to enable the learner to be prepared for the upcoming sprints.

&#x20;   Chatgpt is being used as the AI companion to take care of generating the weekly targets as well as the detailed

&#x20;   daily tasks.  Every day, evidence of work is required. Similar to actual software development, after the end of 

&#x20;   each sprint, a retrospective is performed.  Planning for the next week happen right after the retrospective.
