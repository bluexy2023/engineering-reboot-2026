# Engineering Reboot 2026
A structured 26-week engineering reboot program.

## Goals
- Rebuild software engineering skills
- Develop modern development workflow
- Build portfolio projects
- Establish engineering discipline
- Prepare to enter into software development to develop EB2-NIW endeavor



# Project Objective:

  - Get back to software engineering, leveraging 22+ years of industry experience, 8 years actually 

    spent on software development, while the rest doing project management, and software engineering

    management.  This is a 6-month long self-directed activity which covers essential software engineering

    principles as well as using AI as a companion for software engineering.

  - The study sessions cover various topics which should enable one to bet back to software engineering and become

    up-to-date to the latest development practices after a long period of non-practice

  - The 6-month period is divided into 6 months of 4 weekly sprints per month, with each week taking 5 hours. 

    Daily tasks are prepared one week in advance to enable the learner to be prepared for the upcoming sprints.

    Chatgpt is being used as the AI companion to take care of generating the weekly targets as well as the detailed

    daily tasks.  Every day, evidence of work is required. Similar to actual software development, after the end of 

    each sprint, a retrospective is performed.  Planning for the next week happen right after the retrospective.
