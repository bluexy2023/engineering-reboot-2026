•  What is a function?
- a function is a code block that defines a reusable unit of program
  which can be called repeatedly, which behaves the same way across executions.  It defines a name, the parameters being fed to it, and its return value
• Why use functions?
- functions enables code reuse. For example, a program calls for repeatedly 
  converting a number to float(), a float() function was defined internally.
  A software programmer/developer can develop/define their own functions for code that would be performed repeated. Ex. sum, difference, converting kilometers to miles, etc
• What is a parameter?
- a parameter is the 'input' to the functions.  it could be zero or multiple, which is defined by the names assigned to them when defining a function
• What is a return value?
- a return value is the value that we return to the caller. it is performed by the keyword 'return' within a function definition.  once this line is encountered, the function exits with the return value defined
• How does code reuse improve software?
- it allows for code sharing across different modules, which allows for better readbility as well as keep the software size (in terms of number of lines of code to be manageable).  It also allows for defining a single responsibility for a software piece, which allows for a more efficient of managing the software code
