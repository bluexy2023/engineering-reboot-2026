# Day 03 Journal
## What I Learned
1. Master branch - where all the files reside -- the root directory of sorts of this repository of files
2. Staging branch - when you have changed something and added those changes to the master branch -- they will remain in the staged branch until a 'commit' is performed
3. newly added files would appear with 'red' color until 'git add' is performed on those specific files or folders/directories
4. git log gives you the history of commits to the repository along with its hash information to show that every commit is unique
## Commands Practiced
- git status
- git add
- git commit
- git log
