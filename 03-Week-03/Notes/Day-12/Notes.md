- Today's exercises focused on list contents traversal
- It is a method of accessing each member or content of a list in a serial manner using a form of a looping construct
- A for loop work in a way that it allows for list traversal by using the form, "for tempVar in listVar".  It puts the existing value of member in the list at a certain index, increments the index, then continues the loop until the index becomes list length - 1
• Why are loops important?
- loops are important in that it allows for automating accessing array or list members without having to manually accessing each member in the collection using explicit indexes.  It allows for simplifying recurring execution of index-based data
• How are loops used in software systems?
- software systems use loops for parsing data, traversing an array, reading CSV files, as well as symplifying configuration information
