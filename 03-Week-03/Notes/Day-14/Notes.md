# Key Observations
- collection processing is the act of processing a group of data in a list, by traversing each one
- aggregation is the treatment of a collection as a single value, ex. min(), max(), sum().  this sums up, or aggregates the collection into a single value or meaning
- sum() works by getting the total of all member values in a list
- len() works by counting the number of elements in a collection or list
- records can be counted using loops by using a counter variable.  Alongside an if statement, using loops for collection becomes very powerful in that each value can be evaluated against certain constraints or conditions
- collection processing might appear in real-world software where a report is needed to summarize data to get statistics from it.  Also, in parsing and analyzing individual elements, certain values that stand out can be called out.
